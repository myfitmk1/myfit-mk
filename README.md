
# MyFit MK - Владо Смилевски

Професионална фитнес апликација развиена со React, TypeScript и Tailwind CSS.

## Функционалности

- **Планер:** Креирање и следење на тренинзи.
- **Вежби:** База на податоци со 200+ вежби, категоризирани по мускулни групи.
- **Календар:** Преглед на закажани и завршени тренинзи.
- **Статистика:** Анализа на напредокот.
- **Галерија:** Следење на визуелна трансформација.
- **Gemini AI:** Интеграција за генерирање планови и совети.

## Инсталација и Стартување

1. **Клонирајте го репозиториумот:**
   ```bash
   git clone https://github.com/myfitmk1/MyFit-MK-demo.git
   cd MyFit-MK-demo
   ```

2. **Инсталирајте зависности:**
   ```bash
   npm install
   ```

3. **Стартувајте ја апликацијата:**
   ```bash
   npm run dev
   ```

## Градба за продукција (Build)

```bash
npm run build
```
